Vignette for Spark on Titanic Data


Environment required: 

Virtual Machine provided for Spark.
Download Here ( https://buffalo.box.com/s/t72vy6py6r6v852anwa55whl4mmibfaw )



The vignette code is kept in the file vignette.scala. To run it, you can do the following

1. Copy the vignette.scala file and titanic.csv files to a folder of your choice
2. From inside the folder, open the terminal and run 'spark-shell'
3. Once inside spark-shell, use the following command to load the scala file and execute it

:load vignette.scala


You will be able to see the output on the screen.
